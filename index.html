My Projects

  • Topic Modeling with Latent Dirichlet Allocation (LDA)

    Implemented the collapsed Gibbs Sampler for LDA inference using 20 Newsgroups dataset and compared the LDA topic representation to a bag-of-words representation with respect to how well they support document classification. Generally, the bag-of-words model performs well but for smaller datasets, LDA performs better than the bag-of-words model.

  • Experiments with Bayesian Linear Regression

    Investigated the effect of number of examples, the number of features, and the regularization parameter on the performance of the algorithm. The performance of the algorithm increases with increase in the number of examples, stays almost the same with increase in number of features and decreases with increase in both number of features and examples.

  • Part-of-Speech Tagging using Neural Networks

    Implemented using Multilayer Perceptron Network - A supervised learning model built on the Penn treebank corpus with vectorized core features of prefixes, suffixes, previous, and next word for each word in each sentence of the corpus. A 97% accuracy for the model was achieved.

  • Dialog Act Classification using Neural Networks

    Used Neural Networks for the classification of text to identify the goal of the Dialogue Act in a conversation using The Switchboard Dialog Act Corpus with an accuracy of 96%.

  • Image Orientation Classification

    Implemented the K-nearest neighbor, AdaBoost and Neural Network algorithms on the same dataset. The results show that the Neural Network model surpasses with 76% transcending both K-nearest neighbor and AdaBoost with 71% and 70% accuracy respectively.

  • Object Classification from images using Convolutional Neural Networks

    Trained a Convolutional Neural Network using CIFAR-10 dataset for object detection and classification from images. The network was built using Tensorflow and CUDA. A training accuracy of 97.33% was achieved with 82% test accuracy.

  • Text Classification using Convolutional Neural Networks and Recurrent Neural Networks

    Trained a Convolutional Neural Network and a Recurrent Neural Network on IMDB dataset for binaryclassification of Movie Reviews and performance comparison between the two networks. While both themodels give a test accuracy of around 80%, the CNN overfits on smaller datasets while the RNN has much more training complexity
